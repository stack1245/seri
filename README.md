# Seri - 임베드 빌더 봇

Discord 서버에서 아름다운 임베드를 쉽게 만들고 관리할 수 있는 봇입니다.

## 기능

- **임베드 생성**: 직관적인 모달과 버튼을 사용하여 임베드 생성
- **커스터마이징**: 제목, 설명, 색상, 필드 등을 자유롭게 조정
- **저장 & 불러오기**: 만든 임베드를 저장하고 나중에 다시 사용
- **미리보기**: 임베드가 어떻게 보일지 미리 확인
- **JSON 내보내기**: 임베드 데이터를 JSON 형식으로 내보내기
- **전송**: 저장된 임베드를 여러 채널에 전송

## 명령어

### `/create`
새로운 임베드를 생성합니다.

1. 제목(선택), 설명, 색상을 입력하는 모달이 나타납니다
2. 버튼을 사용하여 추가 기능을 이용할 수 있습니다
   - **제목 추가**: 임베드 제목 설정
   - **필드 추가**: 제목과 내용이 있는 필드 추가 (최대 25개)
   - **색상 변경**: 임베드 색상 변경
   - **미리보기**: 완성된 임베드 미리보기
   - **저장**: 임베드를 이름과 함께 저장
   - **완료**: 임베드 생성 완료

### `/list`
저장된 임베드 목록을 확인합니다.

- **불러오기**: 저장된 임베드를 선택하여 불러오기
- **삭제**: 저장된 임베드를 삭제

### `/load <name>`
특정 임베드를 불러옵니다.

- 임베드 이름을 인자로 지정하여 바로 불러오기 가능

## 색상 옵션

기본 제공 색상:
- RED (빨강)
- GREEN (초록)
- BLUE (파랑)
- YELLOW (노랑)
- PURPLE (보라)
- CYAN (청록)
- GRAY (회색)
- DARK_GRAY (진회색)

또는 16진수 색상 코드를 입력할 수 있습니다 (예: `0xFF0000`)

## 설치 및 실행

### 요구사항
- Python 3.10 이상
- py-cord
- python-dotenv

### 설치
```bash
pip install -r requirements.txt
```

### 설정
`.env` 파일을 생성하고 Discord 봇 토큰을 입력합니다:
```
DISCORD_TOKEN=your_token_here
```

### 실행
```bash
python main.py
```

## 파일 구조

```
seri/
├── main.py                 # 봇 메인 클래스
├── commands/
│   ├── create.py          # 임베드 생성 명령어
│   └── manage.py          # 임베드 관리 명령어
├── utils/
│   ├── constants.py       # 상수 정의
│   ├── data_manager.py    # 데이터 관리
│   ├── extension_loader.py # 명령어 로더
│   ├── graceful_shutdown.py # 안전한 종료
│   └── logging_config.py  # 로깅 설정
└── data/
    └── embeds.json        # 저장된 임베드 데이터
```

## 저장 데이터 형식

임베드는 JSON 형식으로 저장됩니다:

```json
{
  "title": "임베드 제목",
  "description": "임베드 설명",
  "color": 3498843,
  "fields": [
    {
      "name": "필드 이름",
      "value": "필드 내용",
      "inline": false
    }
  ],
  "author": null,
  "footer": null,
  "image": null,
  "thumbnail": null
}
```

## 주의사항

- 임베드는 최대 25개의 필드를 포함할 수 있습니다
- 필드 이름은 최대 256자까지 가능합니다
- 필드 값은 최대 1024자까지 가능합니다
- 설명은 최대 4096자까지 가능합니다
